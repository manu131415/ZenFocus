# 🧘 ZenFocus

> Complete Tasks. Earn XP. Stay Focused.

ZenFocus is a gamified productivity platform designed to help users stay focused, manage tasks efficiently, and build productive habits through an engaging XP-based reward system.

Unlike traditional to-do applications, ZenFocus motivates users by rewarding task completion with Experience Points (XP), making productivity feel like progress in a game.

---

## 📖 Overview

Students and professionals often struggle with procrastination, lack of motivation, and poor task management. ZenFocus addresses these challenges by combining task organization with gamification mechanics.

Users can create tasks, track their progress, complete activities, and earn XP points that encourage consistency and long-term productivity.

## 🎮 XP System

ZenFocus rewards users for staying productive.

| Action | Reward |
|----------|----------|
| Complete a Task | +10 XP |
| Complete Multiple Tasks | Bonus XP |
| Maintain Productivity | Increased Progress |

The XP system helps users stay motivated and transforms productivity into a rewarding experience.
